makeAQuiz({
elementId: '#myQuiz',
questionsArr: [{title: 'Where is this place?',
                options: ['PopQuiz',
                          'Canada',
                          'Random Place'],
                answer: 'PopQuiz'}
                {title: 'Are you a csc309 student?',
                options: ['Yes',
                           'No',
                           'I don't know'],
                answer: 'Yes'}],
time: {minutes: 0,
       seconds: 50}});
Settings Parameters
Parameter Description Type Required
elementId ID of a div container given created and given by the user String Yes
questionsArr User questions data Array of JSON, which each JSON desribes a question Yes
time The time limit determined by the user JSON Yes
Question Parameters
Parameter Description Type Required
title Questions description/title String Yes
options Options of the question determined by the user Array of strings Yes
answer Answer to the question determined by the user String Yes
Time Parameters
Parameter Description Type Required
minutes minutes protion of the total given time to complete the quiz Integer Yes
seconds seconds protion of the total given time to complete the quiz Integer Yes
