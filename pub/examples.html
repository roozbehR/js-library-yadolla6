The makeAQuiz API takes a JSON that has 3 fields (key-value pairs), first field is elementId, second field is quesionsArr, and third field is time. Following is the doc provided to explain what each field is.(Please refer to the alpha realease report API picture to understand this part)

Fields Description of the Field Type of the Field Required
elementId ID of a div container created and given by the user before hand which is fed to the api for the use of the libary String Yes
questionsArr User questions data that is fed to the API, in this array there are JSON objects that represenet each questoin that the user would like to put in the quiz (Refer to the report submitted on Quercus) Array of JSON, which each JSON desribes a question Yes
time The time limit determined by the user, which is a JSON with two fields whic represenet minutes and seconds (Refer to the reprot submitted on Quercus) JSON Yes
